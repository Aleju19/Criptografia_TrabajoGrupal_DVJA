# Política de Privacidad

## Objetivo

Establecer los lineamientos para el tratamiento de los datos personales recopilados por la aplicación DVJA, garantizando el cumplimiento de la Ley Orgánica de Protección de Datos Personales (LOPDP) del Ecuador.

---

# Responsable del tratamiento

Aplicación: DVJA (Damn Vulnerable Java Application)

Responsable: Administrador del Sistema

---

# Datos personales recopilados

La aplicación almacena los siguientes datos personales durante el registro de usuarios:

- Nombre completo
- Nombre de usuario
- Correo electrónico
- Contraseña (almacenada mediante BCrypt)

---

# Finalidad del tratamiento

Los datos personales son utilizados exclusivamente para:

- Registro de usuarios.
- Autenticación.
- Gestión de cuentas.
- Control de acceso basado en roles.
- Administración del sistema.

Los datos no serán utilizados con fines comerciales.

---

# Medidas de seguridad implementadas

Para proteger la información de los usuarios se implementaron los siguientes controles:

- Hash BCrypt para el almacenamiento seguro de contraseñas.
- Cifrado AES para proteger el correo electrónico.
- Control de Acceso Basado en Roles (RBAC).
- Validación de autenticación mediante sesiones.

---

# Conservación de los datos

Los datos permanecerán almacenados únicamente mientras la cuenta del usuario permanezca activa.

Cuando una cuenta sea eliminada, la información deberá eliminarse conforme a las políticas institucionales.

---

# Derechos del titular

Los usuarios podrán ejercer los siguientes derechos:

- Acceso
- Rectificación
- Actualización
- Eliminación
- Oposición al tratamiento de sus datos

---

# Base legal

Ley Orgánica de Protección de Datos Personales (LOPDP).
